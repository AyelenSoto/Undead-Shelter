
Si no les aparece el env aqui una copia, solo lo crean fuera de todos los proyectos, osea en la carpeta "UNDEAD-SHELTER"
EL NOMBRE DEL ENV DEBE SER "  .env  " con el punto que tiene tambien y pegan lo que esta debajo de la linea separadora ahi abajo.

Despues tienen que instalar Composer desde el cmd del Visual Studio con el siguiente comando:

composer install

Tambien para el npm, tienen que tener instalado NODE JS y luego ejecutan este comando:

npm install

[Si no les deja ejecutar, usen un command promt en vez de powershell]

Luego generan la clave de seguridad para el ENV con este comando: 

php artisan key:generate

¡¡¡¡¡para ello usen el Comand Promt del mismo Visual Studio!!!!!

- Nota: NO MODIFIQUEN EL .ENV
- Nota2: Este env de ahi abajo es uno limpio osea antes de tener la clave generada.
